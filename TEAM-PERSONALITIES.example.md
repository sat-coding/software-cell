# Team Personalities

Use this file inside a project to declare which Agency Agents personalities power the software cell.

## Default mapping

- Coordinator: `product/product-manager.md`
- Engineer: `engineering/engineering-rapid-prototyper.md`
- Verifier: `testing/testing-reality-checker.md`
- Designer: `design/design-ui-designer.md`
- Runtime: `engineering/engineering-devops-automator.md`

## Notes

- Keep the team small.
- One personality per role is enough to start.
- Change personalities only when the project phase changes.
- Personality should affect standards and outputs, not just tone.
