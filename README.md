# Software Cell Gist

A gist-style writeup for the pattern of building products with a small persistent LLM software team.

## Files

- `software-cell.md` — the main gist draft
- `AGENCY-INTEGRATION.md` — how to use Agency Agents as the personality layer
- `TEAM-PERSONALITIES.example.md` — starter mapping file for projects

## Intent

This is written in the spirit of Karpathy's LLM wiki gist, but for a different pattern:

> not an LLM-maintained wiki, but an LLM-run software team with runtime validation.

Use it as a seed document for future project/system design.
